CSCE 242 Web Applications
K. Colson Carey

This class teaches different coding languages, namely HTML, CSS, Javascript
used to create a functional and aesthetically pleasing website. Students
will be expected to complete many different assignments from developing
the frontend of a website using HTML and CSS to the backend with Javascript.

Assignments

Assignment02

This assignment used basic html alongside basic css to implement styles and pictures into the design of the website.

Assignment03

Assignment04

This assignment brings together all of the frontend development experience from the past 3 weeks to replicate the USC College of Arts and Sciences homepage.

Assignment05

Assignment06
